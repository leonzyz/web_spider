                          <p><br/><br/><br/><br/><br/><br/>简单来讲，自己整理的这套图鉴主要针对普通的多肉植物爱好者：<br/><br/>1、主要用于大家入门认识，浇水及日照等养护方法仅供参考。<br/><br/>2、简单易懂直观、方便保存及查阅（推荐大家把图片保存后，更改图片名为多肉植物的名称）<br/><br/>3、图鉴内图片全部为自己拍摄，并且也是自己正在养着的多肉植物。<br/><br/>4、内容上主要是将自己平时的养护经验与一些多肉植物特点标记出来。（博主所在地区：山东威海）<br/><br/>5、因为自己也不是专业级别的，所以难免会有错误，如果大家发现可以微博@我，或者私信留言，我会及时查阅更改的。<br/><br/><br/> <br/><br/> <br/><br/>【多肉植物常见问题解答】<br/><br/>因为图片内容有限，有很多重复的信息我都没有添加上去，这里统一解答下（一次整理不完,持续更新）：<br/><br/> <br/><br/>多肉植物的生长期与休眠期是什么时候？<br/><br/>答：春秋季节是多肉植物的生长期，夏冬季节为休眠期，但是又因地域及气候条件，某些地区夏季与冬季也是可以生长的。比如：夏季开空调，冬季北方地区有暖气，都可以使多肉植物继续生长。<br/><br/> <br/><br/>“全日照”指的到底是什么？<br/><br/>答：“全日照”指的是日照时间，表示某些多肉植物可以全天接受日照，并不是指的完全放在阳光下沐浴，夏季还是需要采取一些防晒措施的（防晒网）。<br/><br/> <br/><br/> <br/><br/>多肉植物适宜生长的温度是多少呢？夏季与冬季多少度又会休眠？<br/><br/>答：对于多肉植物来说，最适宜的生长温度是10℃-30℃。当然，温度不可能是一个恒定值。夏季超过35℃时，部分多肉植物会进入休眠状态，特别是“冬种型”（冬季生长），当温度达到40℃时，几乎所有的多肉植物都会进入休眠状态，即使“夏种型”（夏季生长），也会进入休眠状态（不再吸收水分）。冬季在低于5℃时，部分多肉植物会进入休眠状态。0℃是冰点，水分会慢慢结冰，多肉植物叶片和枝干内部大部分都是水分，所以当低于0℃时，多肉植物内部的水分也会慢慢结冰，植物主体肯定都不会再生长了，这时候就一定要断水。<br/><br/> <br/><br/>图鉴内是否能加入多肉植物所适合的栽培介质？<br/><br/>答：这个在首页置顶博文里已经写过了，从目前自己使用的介质来看，松软的泥炭土+颗粒是最为合适的。我常用的靓土成分也是：泥炭土+稻壳+珍珠岩+蛭石等，为了省事，所以每次直接用靓土栽培。并且几乎家里所有的多肉植物全部采用统一质料。<br/><br/> <br/><br/>夏季断水是否就是完全不浇水？这个时间大概要持续多长？<br/><br/>答：常说的夏季断水其实也只是针对部分多肉植物而言，植物总还是需要水分的，所以在夏季高温时，浇水间隔延长至一月一次，少量最佳。多少让土壤中保持一丁点的水分。野外中生存的多肉植物在夏季也是依靠地上土壤中的水分度夏，所以可以持续几个月甚至半年不用雨水也能存活。但是现在把多肉植物移种到家里，红陶一类的花器很容易就会完全干透，在夏季有时候我还继续保持一周甚至5天浇水一次。持续时间主要看天气与温度状况，发现秋季来临，温度开始慢慢降低就可以加大浇水量了。<br/><br/> <br/><br/>是否能在图鉴里标明“冬种型”（冬季生长）或“夏种型”（夏季生长）？<br/><br/>答：考虑到现有资料较少，有许多多肉植物很难分辨到底属于哪一种，只有自己亲自养殖观察后才能得出结论，所以没有直接添加在图鉴里，不过现有资料中能确定的我肯定会添加进去。“冬种型”与“夏种型”在栽种过程中并没有太大区别，生长季节依然是春秋季，夏季与冬季依然也会休眠，所以对于普通爱好者来讲，只要知道如何养护即可~ 具体可参考第④和⑥条疑问解答。<br/><br/> <br/><br/>看图鉴上所有肉肉的浇水和日照需求都不一样，那可以混在一起栽吗？那些组合混栽的怎么养护？<br/><br/>答：可以混栽！而且组合起来的效果更美~ 不同颜色的肉肉搭配在一起看起来更漂亮。养护方面正常日照、浇水即可，我觉得混栽的肉肉长得更好更快，也许是共生长在一起有竞争的感觉吧！但是要特别注意一点：混栽组合多肉植物尽量选择同一科属栽种，因为习性大致相同，所以管理起来也比较方便。例如：石莲花一类混植、景天属一类混植、也可以将景天属与石莲花属一起混植，因为都同属于景天科。不过番杏科和景天科混植在一起就不太好了，因为番杏对水比较敏感，时间长了就会发现，还得挖出来重新搭配。<br/><br/> <br/><br/>什么是“缀化”？<br/><br/>答：缀化是指花卉中常见的畸形变异现象。<br/><br/>　　缀化变异是指某些品种（种植）的多肉植物受到不明原因的外界刺激，其顶端的生长锥异常分生、加倍，而形成许多小的生长点，而这些生长点横向发展连成一条线，最终长成扁平的扇形或鸡冠形带状体。缀化变异植株因形态奇异，观赏价值更高，又因其稀少，较原种（种植）更为珍贵。<br/><br/></p>
              </div>
